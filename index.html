📝 Куда писать JavaScript-код?

🧪 1. Консоль браузера

Мгновенная проверка: F12 → вкладка Console

📁 2. HTML-файл (<script>)

Код внутри тега script

<script> alert("Hello"); </script>

🗄️ 3. Внешний .js файл

Подключение через src

<script src="app.js"></script>

⭐ Совет: используй VS Code

💾 Как сохранять код и запускать?

  • VS Code / Sublime Text → создай файл с расширением .html или .js.
  • Сохраняй через Ctrl+S (Cmd+S) — важная привычка.
  • HTML + JS: дважды кликни по index.html → откроется в браузере.
  • Node.js: сохрани script.js, запусти через node script.js.
// Файл main.js
let userName = "Елена";
console.log("Привет, " + userName);

⚠️ После изменений сохраняй файл и перезагружай страницу (F5).

📌 Комментарии — объясняем код

Комментарии не выполняются, они для людей. Помогают понять логику.

// Однострочный

// Это комментарий
let x = 10; // пояснение

/* Многострочный */

/*
Можно писать
несколько строк
*/

💡 Зачем? Документировать сложные участки, временно отключать код.

let total = 100;
// total = total + 50; ← этот код временно не работает
console.log(total); // Выведет 100

📦 Переменные — контейнеры для данных

Переменная хранит значение (число, текст, true/false). Это как коробка с именем.

let возраст = 25;
let имя = "Артем";
let isStudent = true;

Как создать переменную в JS?

  • let — современный стандарт (значение можно менять).
  • const — константа, нельзя переназначить.
  • var — старый способ (лучше не использовать).
let city = "Москва";
const PI = 3.14159;
city = "Санкт-Петербург"; // ✅ можно
// PI = 3.14; ❌ Ошибка!

✏️ Как называть переменные (примеры)

  • Имя может содержать буквы, цифры, символы $ и _.
  • Не может начинаться с цифры.
  • Регистр важен: myVar и myvar — разные.
  • Используй осмысленные имена: userAge, totalPrice (camelCase).

✅ Правильные имена:

let firstName = "Ольга";
let _private = 123;
let score2025 = 99;

❌ Неправильные имена:

// let 1stPlace = "gold"; // ошибка: начинается с цифры
// let my-name = "js"; // дефис не разрешен

🎯 Живой пример: переменные в действии

// Объявляем переменные
let product = "Ноутбук";
let price = 75000;
let discount = 0.15;
let finalPrice = price - (price * discount);

console.log("Товар: " + product);
console.log("Цена со скидкой: " + finalPrice + " ₽");

price = 82000;
finalPrice = price - (price * discount);
console.log("Новая финальная цена: " + finalPrice);

📢 В консоли браузера (F12):

Товар: Ноутбук
Цена со скидкой: 63750 ₽
Новая финальная цена: 69700 ₽

⚡ let vs const (что выбрать?)

let (изменяемая)

let count = 0;
count = count + 1; // ✅
count = 100; // ✅

Используй, если значение будет меняться.

const (неизменяемая)

const birthYear = 1995;
// birthYear = 2000; ❌ Ошибка!

Идеально для чисел, строк, которые не должны меняться.

🧠 Важно: const защищает от случайного переназначения — меньше багов!

const COLOR_RED = "#FF0000";
let userChoice = COLOR_RED;

📊 Какие значения можно хранить?

  • number — числа: let age = 28;
  • string — текст: let name = "Алиса";
  • boolean — true/false: let isLogged = true;
  • undefined — значение не задано: let x;
  • null — «пусто», намеренное отсутствие.
let price = 99.9; // число
let message = "Hello!"; // строка
let isOpen = false; // булево
let data; // undefined

🔍 Узнать тип переменной через typeof:

console.log(typeof price); // "number"

⚠️ Частые ошибки новичков

  • Не объявили переменную → ReferenceError
  • Использовать переменную до объявления (временная мертвая зона)
  • Путать кавычки — закрывайте строки одинаково
  • Пробелы и регистр — myVariable и myvariable разные

❌ Пример ошибки:

// забыли let/const
city = "Paris"; // плохая практика
// правильно:
let city = "Paris";

✅ Всегда используй 'use strict' или пиши let/const.

🧪 Твой первый скрипт — попробуй!

Открой консоль браузера (F12) и скопируй этот код:

// Создаем переменные о себе
let myName = "Твое имя";
let myAge = 20;
let hobby = "JavaScript";

console.log("Привет! Меня зовут " + myName);
console.log("Мне " + myAge + " лет");
console.log("Я люблю " + hobby);

myAge = myAge + 1;
console.log("В следующем году будет " + myAge);

🎉 Итог: Ты объявил переменные, присвоил значения и вывел в консоль!

✨ Бонус: меняем переменные через prompt()

let userName = prompt("Как тебя зовут?");
let greeting = "Привет, " + userName + "! Добро пожаловать в JS!";
console.log(greeting);
alert(greeting);

✅ Это показывает, как переменные работают с пользовательским вводом.
